한경 산업 분류 전용 수정본
=========================

데이터 흐름
1. KODEX 200(069500) ETF 구성종목 전체
2. KODEX 코스닥150(229200) ETF 구성종목 중 비중 상위 100개
3. 한국경제 데이터센터에서 각 종목코드의 FACTSET 산업 분류 매칭
4. 네이버 증권에서 종가, 등락률, 시가총액만 조회
5. 산업별 시가총액 가중 등락률 계산
6. 상승률 TOP 5 / 등락률 BOTTOM 5 표시

중요
- 네이버 업종 분류는 사용하지 않습니다.
- 한경 JSON 구조 매칭과 렌더링 DOM 그룹 매칭을 각각 수행하고 정상적인 매핑만 선택합니다.
- 산업 수가 비정상적으로 적거나 한 산업에 종목이 몰리면 게시하지 않습니다.
